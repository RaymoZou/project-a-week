# Week 29 of Paw - OpenGL Cube

My second attempt at rendering a cube with OpenGL version 3.3.

## Installation (Arch Linux):
1. Install dependencies with `sudo pacman -Syu sdl2 glew`.
2. Compile using `make`.
3. Run with `./main.exe`.
